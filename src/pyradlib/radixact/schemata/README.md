# File structures and XML schemata
These markdown documents provide an overivew of the file structures, including XML schemata, for Radixact delivery files. 

These files can be found cached on PCs running the Accuray Delivery Analaysis software, in the following location:
```
C:\tomo\da\pts\[URN]
```

These files are also aggregated by the Accuray Patient Data Extractor tool, though filenames may differ due to anonymisation and use of CDMS extensions.
